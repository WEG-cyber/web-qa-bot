# Cellbedell NU1 – G2s 產品規格說明書

Cellbedell NU1 – G2s 是一部高效能的 Edge AI 門禁控制器。

## 1. 核心硬體規格
1. **電源需求**：DC 12V / 2A，具備過電流與反極性保護。
2. **系統架構**：採用 Edge AI 門禁控制器架構與工業級 PCB 主機板。
3. **模組設計**：讀卡頭與主機板分離式設計，支援即插即用 (Plug & Play)。
4. **輸出入介面**：提供繼電器 (NC/NO/COM)、出門開關訊號、系統重設與 WiFi 重設按鍵。
5. **安裝方式**：支援壁掛式 (Wall-mounted) 或表面安裝 (Surface-mounted)，外殼採工業級 ABS 材質。

## 2. 認證與通訊方式
- **行動憑證驗證**：支援 Apple Wallet / Android Wallet (透過 Cellbedell Vkey)。
- **NFC 感應**：支援 ISO14443 Type 2 標準。
- **QR Code**：支援螢幕掃描辨識。
- **藍牙 (BLE)**：支援手機 App 近距離感應。
- **通訊協議**：支援 MQTT / HTTPS RESTful API / AWS / Google Cloud。

## 3. 軟體應用
- **維修與管理**：請洽總工程師 David 或使用 CellHost App 進行診斷。
- **使用者端**：請下載 CellVkey App 進行虛擬鑰匙管理。
